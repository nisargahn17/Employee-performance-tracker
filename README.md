# Employee Performance Tracker

## 📌 Overview
Employee Performance Tracker is an Android application developed using Kotlin and Jetpack Compose to monitor employee attendance, performance, and records through an intuitive dashboard.

## ✨ Features
- Secure Login Screen
- Employee Dashboard
- Employee Management
- Attendance Tracking
- Performance Monitoring
- Interactive Charts and Statistics

## 🛠 Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose
- Firebase Authentication
- Cloud Firestore
- Material 3

## 📱 Screens
- Splash Screen
- Login Screen
- Dashboard
- Employees
- Attendance
- Performance
